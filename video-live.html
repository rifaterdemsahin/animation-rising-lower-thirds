Live Overlay

Type the two lower-third labels and hit Play to preview the spring-rise animation exactly as it will appear as a live overlay (e.g. an OBS Browser Source). The checkerboard behind the cards represents transparency — the real background is fully transparent.

Reusable as a browser-source URL

Skip the form entirely: pass the labels as query params and add &autoplay=1. The page strips its own chrome and plays automatically on load — point OBS/vMix's Browser Source straight at the URL. 


      
Reusable as a JS API

      
Any page can drive the same engine directly:

      
<script src="assets/js/lower-thirds.js"></script>
<script>
  LowerThirds.mount(document.getElementById("stage"), {
    card1: { icon: "💼", text: "Real Jobs" },
    card2: { icon: "👤", text: "Real People" },
  });
</script>
