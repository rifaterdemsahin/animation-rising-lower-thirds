Rising Lower Thirds

A practical animation helper for the twin "Real Jobs / Real People" lower-third cards: two rounded cards that spring up from off-screen, bounce, and settle into place, with the second card staggered 150ms behind the first. The full motion spec lives in problem.md. This particular rising-card animation style is demonstrated in this video — that's the reference for the look and timing this toolbox reproduces.

The problem

problem.md is the formal design/motion spec this project implements: two rounded rectangle cards (dark teal, 85% opacity, 24–30px radius), a cyan icon on the left and serif text on the right, that rise from off-screen and settle with a damped-spring bounce — the second card entering 150ms after the first. It also defines the reference screenshot below and the exact cubic-bezier/damped-harmonic math the animation should follow.

What we're trying to do

Instead of hand-keyframing this animation in a video editor every time the copy changes, you type the text you want for each lower third and the same animation engine (assets/js/lower-thirds.js) renders it for you. One engine, two outputs:

  • video-recorded.html — type your text, render it to a canvas, and export an actual downloadable video file (WebM/MP4) you can drop into an edit timeline.
  • video-live.html — the same animation as a lightweight, transparent-background overlay you can point OBS/vMix at for a live stream, driven by URL query params or the JS API directly.

This is a small toolbox, not just two forms: every page works by clicking through the UI, but every page is also drivable without touching the UI at all — a URL query-param API for scripting/automation, and a plain JS API (LowerThirds / LowerThirdsRecorder) for embedding the engine in your own page. Each page documents both under an "API"/"Reusable" panel.

See recommendation.html for suggested next steps, informed by the video-production tooling evaluated for this project, and MCP_GUIDE.md for driving this toolbox from an MCP client/agent instead of the UI.
