💰 Cost Analysis

What each environment actually needs as input, what it produces, and roughly how long it takes to build (one-time) versus use (per project) — plus the only two places real money changes hands in this whole system. See index.html's "Same animation, three different environments" section for how the outputs differ visually; this page is about effort and cost specifically.

💵 Where money actually gets spent

Everything above is free static hosting and browser-side computation except two things:

  1. Image generation (Gemini "Nano Banana" via fal.ai's nano-banana-2 endpoint, used for backgrounds). Priced per output megapixel, roughly $0.0398/MP — this project's 1920×1080 background is ~2.07MP, so about $0.08 per generation. You only pay when you actually generate a new image; reusing assets/k8s-explainer-background.png (or any background you've already generated) costs nothing.
  2. Fly.io hosting for the API in server/ (lower-thirds-api.fly.dev). The machine is configured to auto-stop when idle and auto-start on request (auto_stop_machines/min_machines_running: 0 in server/fly.toml), so cost scales with actual usage rather than running 24/7 — exact numbers depend on your Fly.io plan and request volume, which this page can't state precisely without checking live billing; check the Fly.io dashboard for your account's current usage.

Everything else — the static site (GitHub Pages), the animation engine, the Builder tools, the standalone HTML export, and driving any of it via the URL API or a browser-automation MCP client — costs nothing to run, no matter how many times you use it.

🤔 Which is cheapest for your situation

  • Reusing existing content (same background, new labels) → free, any environment, seconds.
  • New content, need a video file → 🎥 Recorded Video + a new background (~$0.08, once).
  • New content, need it live → 📡 Live Overlay (no background needed at all — it must stay transparent).
  • New content, need to explain/teach it → 🧭 Explainer + a new content-matched background (~$0.08).
  • No code, want to hand-tweak in Canva → 🎨 the manual path — most generation cost (~$0.30+) and most time (~15–30 min), because nothing is reused/automated.
