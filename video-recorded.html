🎥 Recorded Video

Type the lower-third labels, hit Record, and this page renders the same spring-rise animation — staged three cards deep, over the shared Kubernetes-flow background — to a <canvas>, captures it frame-by-frame with MediaRecorder, and hands you a downloadable video file — no editor round-trip needed. Defaults to the Kubernetes example (same content as explainer.html); change any field to record your own.

🎞️ Output

Download video

Need a different container/codec (e.g. MP4 for an editor that won't take WebM)? The project's ffmpeg-usage skill can convert it — see recommendation.html.

🧰 Use it as a toolbox, not just a form

This page is a thin UI over two reusable APIs — script it instead of clicking through:

🔗 URL API

Pass the labels as query params and add &autorecord=1: the page fills the form, runs the animation, records it, and shows the download link with no interaction needed. 


      
🧩 JS API

      
Call the engine and recorder directly from your own script:

      
<script src="assets/js/lower-thirds.js"></script>
<script>
  var config = LowerThirds.PRESETS.kubernetes; // or your own { cards: [...] }

  // Drive any canvas yourself, frame by frame (6th arg paints a background
  // image before the cards, e.g. LowerThirds.PRESETS.kubernetes.background):
  LowerThirds.drawCanvasFrame(ctx, canvas.width, canvas.height, tMs, config, restY, onBackground);

  // Or reuse this page's recorder end-to-end:
  window.LowerThirdsRecorder.record(canvas, config).then(function (blob) {
    // blob is a video/webm Blob — save it, upload it, pipe it into ffmpeg, etc.
  });
</script>
